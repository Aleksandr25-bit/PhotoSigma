# PhotoSigma - агропромышленный графический фоторедактор

## Общая информация

**PhotoSigma** — узконаправленная программа, разработанная с использованием библиотек PyQt6, предназначенная для удобного синтезирования сорняков на изображения с различными типами культур. Она предоставляет пользователю визуальный холст, где можно загружать фоны, вставлять изображения, изменять их положение, масштаб, яркость, и сохранять результат в различных вариантах.

- **Тип программы:** агропромышленный графический фоторедактор  
- **Платформы:** Windows 10  
- **Язык интерфейса:** Русский

---

## Назначение программы

- Синтезирование изображений культуры с сорностью.
- Обработка изображений для обучения нейронной сети.

---

## Основные функции

### 1. Загрузка изображений и фонов
- Загрузка фонов и вставляемых изображений через интерфейс.
- Просмотр миниатюр изображений в QListWidget.
- Поддержка перетаскивания изображений на холст.

### 2. Изменение яркости
- Увеличение или уменьшение яркости изображения с помощью кнопок или ползунка.
- Фильтрация применяется как к фону, так и к вставленным изображениям.
- Временное изображение удаляется, как и его след в памяти.

### 3. Перемещение, масштабирование и поворот
- **← ↑ → ↓** — перемещение изображения.
- **Q / E** — поворот изображения.
- **Колесо мыши** — масштабирование.

### 4. Undo / Redo (отмена/повтор действий)
- Программа сохраняет состояние холста при каждом изменении.
- Возможность отменять действия (яркость, перемещение, вставку) и возвращать их.

### 5. Сохранение изображений
- Обычное сохранение в файл (.png, .jpg и др.).
- Сохранение сразу 4 или 100 версий изображения с разными углами поворота (90°, 180°, 270° и т.д.).
- Автоматическая генерация имён файлов с индексами.

### 6. Композиция изображений
- Изображения можно вставлять, перемещать по холсту и «заблокировать» нажатием **Enter**.
- Поддержка наложения нескольких изображений.

### 7. Меню "О программе" с открытием документа
- Программа ищет файл `description.docx` в текущей директории и подпапках.
- Автоматически открывает его в Microsoft Word или связанном приложении.

### 8. Кастомный интерфейс
- Безрамочное окно с собственными кнопками закрытия/сворачивания.
- Цветовая палитра: фиолетово-бело-сиреневая.
- Splash screen при запуске (`PhotoSigma.png`).

## Как пользоваться

### 1. Запуск программы
Зайдите в папку, где установлена программа, и запустите исполняемый файл.

### 2. Рабочее окно
Откроется основное окно с двумя библиотеками:
- **Фон**
- **Изображение**

А также вспомогательные функции:
- **Файл**
- **Правка**
- **Увеличить/уменьшить яркость**

### 3. Начало работы
1. **Выберите фон** — укажите папку с фоновыми изображениями.
2. **Выберите изображение** — укажите папку с изображениями для наложения.
3. **Начните композицию**:
   - Перетащите изображение на фон или используйте клавиши **← ↑ → ↓** для позиционирования.
   - Нажмите **Enter**, чтобы зафиксировать изображение на фоне.
   - Используйте **колесо мыши** для масштабирования.
   - Используйте кнопки **Q / E** для поворота.

### 4. Изменение яркости
Используйте кнопки **«Увеличить яркость»** / **«Уменьшить яркость»** для настройки светлоты/контраста.

### 5. Отмена действий
- Используйте кнопку **«Правка»** или комбинации клавиш:
  - **Ctrl+Z** — отмена
  - **Ctrl+Y** — повтор

### 6. Сохранение
- Перейдите в меню **Файл → Сохранить**.
- Выберите папку и укажите количество экземпляров:
  - **4 изображения** — поворот на 90° каждый.
  - **100 изображений** — поворот на 3.6° каждый.

### 7. Готово
Сохранённые изображения можно использовать в своих целях.

## Преимущества

- **Работа с несколькими изображениями** — легко составлять коллажи и композиции.
- **Точная настройка яркости** — гибкая регулировка светлоты/контраста.
- **Undo/Redo** — безопасное редактирование без потери прогресса.
- **Массовое сохранение** — ускоряет генерацию множественных вариаций.
- **Интуитивный интерфейс** — загрузка, предпросмотр, перетаскивание.
- **Встроенное описание** — пользователю не нужно искать документацию отдельно.
- **Автоматический поиск файлов** — работает на разных устройствах без ручного указания путей.
